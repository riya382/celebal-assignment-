# Superstore Sales Analysis 📊

This folder contains the complete assignment for Superstore Sales Data Analysis.

## 📁 Folder Structure:
* `data/`: Contains the raw `superstore.csv` dataset.
* `notebook/`: Contains `analysis.ipynb` with visual data grids.
* `superstore_analysis.sql`: Contains all the SQL queries (Subqueries, CTEs, and Window Functions).
